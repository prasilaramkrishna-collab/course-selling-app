# How to Upload Quiz Questions

## Admin Panel Guide

### Steps to Add Quiz Questions to a Course:

1. **Go to Admin Dashboard**
   - Navigate to `/admin/our-courses`
   - Find the course you want to add a quiz to
   - Click the "Update" button

2. **Find the "Course Quiz Questions" Section**
   - Scroll down to the quiz editor section
   - You'll see a green button "+ Add Question"

3. **Add Quiz Questions**
   - Click "+ Add Question" to add your first question
   - For each question, fill in:
     - **Question Text**: The question itself (e.g., "What is React?")
     - **Options**: Four answer choices (Option 1, 2, 3, 4)
     - **Correct Answer**: Select the correct option from the dropdown
     - **Explanation**: Optional explanation for the answer

4. **Add More Questions**
   - Repeat the process by clicking "+ Add Question" for each new question
   - Questions are auto-numbered (Question 1, 2, 3, etc.)

5. **Remove Questions**
   - Click the red "Remove" button on any question to delete it
   - Remaining questions will be automatically renumbered

6. **Save Your Quiz**
   - Click "Update Course" button at the bottom of the form
   - The quiz is now attached to your course

## Add Course Plan (Optional)

### Step-by-Step:

1. **Find the "Course Plan (Chapters & Topics)" Section**
   - Scroll down to the course plan editor
   - Click "+ Add Chapter"

2. **Add Chapter Details**
   - **Chapter Title**: (e.g., "Introduction to React")
   - **Description**: What students will learn
   - **Duration**: (e.g., "2 hours")

3. **Add Topics to Each Chapter**
   - Click "+ Add Topic" under each chapter
   - Enter topic names (e.g., "Components", "JSX", "Hooks")
   - Remove topics with the red "Remove" button

4. **Add Multiple Chapters**
   - Click "+ Add Chapter" to add more chapters
   - Chapters are auto-numbered (Chapter 1, 2, 3, etc.)

5. **Save Your Course Plan**
   - Click "Update Course" at the bottom
   - Course plan is now saved

## Testing Your Quiz

1. **View Published Course**
   - Go to the student dashboard and find your published course
   - Enrolled students will see:
     - Course details
     - Materials links
     - "📝 Take Quiz" button

2. **Students Taking the Quiz**
   - Click "Take Quiz"
   - Students answer all questions
   - Submit answers
   - Score is calculated automatically
   - If score ≥ 60%: Certificate is generated ✅
   - If score < 60%: Can retake the quiz

3. **View Certificate**
   - Students with 60%+ score can view their certificate
   - Certificate has:
     - Student name
     - Course name
     - Score percentage
     - Unique certificate number
     - Issue date & validity (1 year)
   - Printable as PDF

## Tips & Tricks

✅ **Best Practices:**
- Keep questions clear and concise
- Use 4 distinct options (no duplicates)
- Always select a correct answer
- Add explanations to help students learn
- Test your quiz by taking it as a student

⚠️ **Things to Avoid:**
- Empty question fields
- Missing correct answer selection
- Duplicate or identical options
- Very long or confusing questions

🔄 **Making Changes:**
- You can edit quiz questions anytime
- Just go back to "Update Course" and modify
- Changes apply to new attempts only
- Previous student scores are preserved

## Troubleshooting

### Quiz Questions Not Saving?
- Make sure all required fields are filled
- Check that Correct Answer is selected
- Click "Update Course" button (not Enter key)

### Students Can't See Quiz Button?
- Verify course is published
- Confirm student has purchased the course
- Check browser cache (Ctrl+F5)

### Certificate Not Generated?
- Student must score 60% or higher
- Quiz must be properly saved with correct answers
- Contact support if issue persists
